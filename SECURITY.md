# 安全政策

## 支持范围

本项目仅维护 `main` 分支的最新版本。旧提交、分叉版本以及把服务暴露到局域网或公网后的部署不在支持范围内。

## 报告漏洞

请使用 GitHub 的[私密漏洞报告](https://github.com/mobaile/apple_store_stock/security/advisories/new)提交问题，不要创建公开 Issue。报告中请包含：

- 受影响版本或提交；
- 最小复现步骤与实际影响；
- 操作系统、Python 和浏览器版本；
- 建议修复方式（如有）。

请勿提交真实 Cookie、访问令牌或其他敏感数据。一般功能问题仍可使用公开 Issue。

## 安全边界

这是只监听 `127.0.0.1` 的单用户本机工具，不是可直接部署的远程服务。项目不会持久化浏览器 Cookie，也不会要求 API 密钥。
